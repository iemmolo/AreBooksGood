## Key Decisions
- SSG: Hugo
- CSS: terminal-css (monospace/terminal aesthetic)
- Themes: pistachio (default), matrix, light, dark
- Theme switcher: CSS custom properties + vanilla JS + localStorage
- Hosting: GitHub Pages with GitHub Actions
- Domain: custom domain (TBD, pointed at GitHub Pages)
- Taxonomies: categories + tags (Hugo built-in)
